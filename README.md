# i9y 

i9y is inclusivity. It is the sum total of the things we do to make our work available to the largest number of people.

i9y includes:

 - i18n, l10n --- internationalization and localization
 - a11y --- accessibility 
 - usability
 - empathy
 - responsive design
 - low-bandwidth and offline usage
 - hiring practices
 - affirming language
 - open licensing
